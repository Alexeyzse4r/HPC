$\Huge{Задание}$

Задача: реализовать алгоритм сложения элементов вектора

Язык: C++ или Python

Входные данные: Вектор размером 1 000..1 000 000 значений.

Выходные данные: сумма элементов вектора + время вычисления

Реализация должна содержать 2 функции сложения элементов вектора: на CPU и на
GPU с применением CUDA.

Отчет о проделанной лабораторной работе - это git-репозиторий с исходным кодом
реализации + описание проделанной работы там же в readme.
Необходимо описать реализацию, объяснив, что конкретно было распараллелено и
почему.

Провести эксперименты: получить сумму векторов разных размеров (провести 5 или
более экспериментов), посчитать ускорение. Результаты привести в виде
таблицы/графика.



$\Huge{Работа}$

Лабораторная работа реализована на C++.

Сложение элементов вектора на CPU(cpu_sum) выполняется стандартно с помощью одного цикла for. Каждый элемент прибавляется к промежуточной сумме.

Сложение элементов вектора на GPU(gpu_sum) выполняется с испольльзованием CUDA. Вычисления распределяются между потоками: вектор делится на блоки, сумма которых подсчитывается параллельно, после этого каждый блок отправляет свое значение(сумму), далее все они складываются уже на хосте.

Для получения достоверных результатов каждый тест(размерность вектора) выполняется 30 раз(NUM_RUNS) и время выполнения усредняется.

Далее в таблице и в виде графиков представлены результаты выполнения эксперимента на различных размерностях вектора от 1000 до 100000000.
